\doxysubsection{Detailed Description}
Represents a point in a 2D space. 

The {\ttfamily \doxylink{structgnode_1_1Point}{Point}} struct contains two floating-\/point coordinates, {\ttfamily x} and {\ttfamily y}, that specify the location of the point in a 2D space. It provides constructors to initialize the point with default or specified coordinates.
