\doxysubsection{Detailed Description}
Abstract base class representing generic data with type information. 

The \doxylink{classgnode_1_1BaseData}{Base\+Data} class provides a common interface for handling data of different types. It stores a type identifier as a string and requires derived classes to implement a method to retrieve a pointer to the stored value.
