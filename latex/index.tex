{\bfseries{GNode}} is a lightweight and extensible C++ framework for building node-\/based graphs. It allows you to define nodes, connect them through input/output ports, and process data through a directed graph structure. GNode is designed to be\+:


\begin{DoxyItemize}
\item {\bfseries{Modular}}\+: you create only the nodes you need
\item {\bfseries{Type-\/safe}}\+: strong typing of ports and attributes ~\newline

\item {\bfseries{Extensible}}\+: supports custom types as long as they are copyable or movable ~\newline

\end{DoxyItemize}

A graph in GNode is composed of\+:


\begin{DoxyItemize}
\item {\bfseries{Nodes}}\+: units of logic or data ~\newline

\item {\bfseries{Ports}}\+: typed inputs/outputs on nodes ~\newline

\item {\bfseries{Links}}\+: connections between ports ~\newline

\item {\bfseries{Graph (supervisor)}}\+: manager for node creation, registration, and update ~\newline

\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{index_autotoc_md2}
