{\bfseries{GNode}} is a lightweight and extensible C++ framework for building node-\/based graphs. It allows you to define nodes, connect them through input/output ports, and process data through a directed graph structure.

{\bfseries{GNode}} is designed to be\+:
\begin{DoxyItemize}
\item Modular\+: you create only the nodes you need
\item Type-\/safe\+: strong typing of ports and attributes
\item Extensible\+: support custom types as long as they are copyable or movable
\end{DoxyItemize}

A graph in GNode is composed of\+:
\begin{DoxyItemize}
\item Nodes\+: units of logic or data
\item Ports\+: typed inputs/outputs on nodes
\item Links\+: connections between ports
\item Graph (supervisor)\+: manager for node creation, registration and update
\end{DoxyItemize}\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Usage Example}{Usage Example}}\label{index_usage_sec}
