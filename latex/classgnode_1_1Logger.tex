\doxysubsection{Detailed Description}
Manages logging functionality using the spdlog library. 

The {\ttfamily \doxylink{classgnode_1_1Logger}{Logger}} class provides a singleton instance of a logger that can be used throughout the application. It utilizes the spdlog library for logging messages to the standard output with color support.

The class is designed to be a singleton, ensuring that there is only one instance of the logger throughout the application. It provides a static method to access this instance.
